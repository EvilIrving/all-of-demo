## 用户中心对外接口目录


* 该目录存放对外定义的接口，服务消费方根据业务自行实现该目录下的接口
* 注意，目录下的接口请按照不同模块自行建包进行划分，示例如下：
    * 身份认证模块接口存放于authentication目录下
    * 用户信息维护模块接口存放于user目录下
    * ...
